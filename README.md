# 🐍 Python + DSA শেখার হাব (বাংলায়)

শূন্য থেকে **FAANG ইন্টারভিউ** পর্যন্ত — সম্পূর্ণ বাংলায় Python ও Data Structures & Algorithms শেখার একটি ইন্টারঅ্যাক্টিভ ওয়েবসাইট। প্রতিটি কনসেপ্টের বিস্তারিত ব্যাখ্যা, চলমান কোড উদাহরণ এবং Time Complexity সহ।

- 🎬 **ইন্টারঅ্যাক্টিভ Sorting Visualizer** (৫টি অ্যালগরিদম, লাইভ compare/swap গণনা)
- 📝 প্রতিটি টপিকে সমজাতীয় **LeetCode প্র্যাকটিস লিংক** + সমাধানসহ প্র্যাকটিস পেজ
- 🔍 লাইভ সার্চ, 🌙 dark/light থিম, 📋 কোড copy বোতাম, 📱 মোবাইল-ফ্রেন্ডলি
- প্রতিটি কোডে বাংলা ব্যাখ্যা ও Time/Space Complexity
